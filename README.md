# Etch-a-Sketch
-It is a Skecth Pad Project, where you can generate some cool drawings and show your creativity.
-Generates 16*16 Grid by default.
-You can input your specific number and it would generate the new grid(input should be between 1 and 100[including 1 and 100]).

# Tech Used:
-HTML for basic structure.
-CSS for UI coloring and flexbox properties.
-JavaScript for DOM manipulation and hovering logic.
